# 软件工程师学习树莓派开发教程

## 介绍

面向软件工程师的树莓派教程，适合以下读者：

* 对物联网感兴趣
* 有一定的软件开发基础
* 熟悉Unix/Linux
* 了解python
* 无电子基础
* 无硬件开发基础
